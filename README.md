**Advanced Encryption Tool**
A simple Python-based GUI application to encrypt and decrypt files securely using AES-256 encryption.

**Features**
1.File Encryption using password
2.File Decryption using the same password
3.AES-256 security with a user-friendly GUI

**How to Run**
Open Terminal / Command Prompt
Navigate to the project folder
Run:
python advanced_encryption_tool.py
